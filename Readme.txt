# PetStoreKarateChallenge

Este es un proyecto de automatización de pruebas utilizando Karate para probar la API de PetStore. El objetivo es demostrar cómo automatizar pruebas de servicios REST utilizando Karate y cómo ejecutar estas pruebas en un entorno controlado.

## Requisitos

- Java JDK 8 o superior
- Maven (para construcción y ejecución de pruebas)

## Clonar el Proyecto

Puedes clonar este repositorio usando el siguiente comando Git en tu terminal:
